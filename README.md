# 🚀 ProdutoAPI

API REST desenvolvida em **ASP.NET Core 8** utilizando boas práticas de desenvolvimento e arquitetura em camadas.

## 📌 Tecnologias

- ASP.NET Core 8
- Entity Framework Core
- MySQL
- JWT Authentication
- Swagger (OpenAPI)
- LINQ
- Git
- GitHub

---

## ✅ Funcionalidades

- CRUD de Produtos
- CRUD de Categorias
- Paginação
- Filtro por Nome
- Ordenação
- Login JWT
- Autenticação via Bearer Token
- Arquitetura utilizando Services e DTOs

---

## 📁 Estrutura do Projeto

```
ProdutoAPI
│
├── Controllers
├── Data
├── DTOs
├── Models
├── Services
├── Migrations
├── Program.cs
└── appsettings.json
```

---

## ▶️ Como executar

```bash
git clone https://github.com/saddanworkdev-droid/ProdutoAPI.git

cd ProdutoAPI

dotnet restore

dotnet ef database update

dotnet run
```
