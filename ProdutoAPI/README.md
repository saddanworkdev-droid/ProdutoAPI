# ProdutoAPI

API REST desenvolvida em ASP.NET Core utilizando Entity Framework Core e MySQL.

## Tecnologias

- ASP.NET Core
- Entity Framework Core
- MySQL
- Swagger
- LINQ
- DTO
- Services

## Funcionalidades

- Cadastro de Produtos
- Cadastro de Categorias
- Relacionamento Produto x Categoria
- Busca de produtos por nome
- Verificação de existência de produtos
- Resumo de produtos
